# 📱🔐 QR Code Malicious URL Detector

## 📌 Overview

This project is a cybersecurity application that scans QR codes and analyzes the extracted URLs to detect potentially malicious or harmful links.

With the increasing use of QR codes in daily life, they have become a target for cyber attacks such as phishing and malware distribution. This tool helps users stay safe by verifying the security of scanned links before accessing them.

---

## ⚙️ Features

* QR code scanning and decoding
* URL extraction from QR codes
* Malicious link detection
* Basic security analysis (patterns, domains, phishing indicators)
* User-friendly safety result (Safe / Suspicious / Malicious)

---

## 🛠️ Technologies Used

* Python
* QR Code Libraries (e.g., pyzbar / qrcode)
* URL Analysis Techniques
* Cybersecurity Concepts

---

## 🧠 How It Works

1. User scans a QR code
2. The system decodes the embedded URL
3. The URL is analyzed using security checks
4. The system classifies the link as Safe or Malicious
5. The result is displayed to the user

---

## 🎯 Objective

The main goal of this project is to enhance cybersecurity awareness and protect users from hidden threats inside QR codes.
